# Disease Prediction using Machine Learning

## Project Description
This project predicts whether a patient has diabetes or not using Logistic Regression.

## Dataset
Pima Indians Diabetes Dataset

## Algorithm Used
Logistic Regression

## Libraries
- pandas
- numpy
- scikit-learn
- joblib

## Accuracy
74.67%

## Author
Aiman Abidi

## Features
- Predicts Diabetes using Logistic Regression
- Takes user input
- Displays prediction result
- Loads a pre-trained model

## Project Enhancements
- Added Graphical User Interface (GUI) using Tkinter
- User-friendly input fields
- Predict button for instant prediction
- Machine Learning model integration
- Real-time diabetes prediction
